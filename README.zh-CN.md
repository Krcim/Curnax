«统一媒体资源平台»

Curnax 是一个开源的媒体资源管理平台，用于统一管理图片、视频、文件等资源，并通过统一链接进行访问与分发。

为什么开发 Curnax

随着对象存储、CDN 和多平台部署的普及，媒体资源往往分散在不同服务之中：

- 图片存储在 OSS
- 视频存储在 S3
- 网站使用 CDN 加速
- WordPress 管理媒体库

当存储迁移或架构调整时，资源链接容易失效，维护成本不断增加。

Curnax 希望提供一个统一的资源管理层，让资源存储、访问和分发更加简单。

核心目标

统一管理

统一管理：

- 图片
- 视频
- 文档
- 静态资源

存储解耦

支持多种对象存储：

- AWS S3
- Cloudflare R2
- MinIO
- 阿里云 OSS
- 腾讯云 COS

链接稳定

更换存储服务时，不影响外部访问链接。

开放扩展

支持：

- API
- 插件
- WordPress
- 自定义存储

规划功能

- 资源管理
- 多存储支持
- 统一访问链接
- CDN 集成
- WordPress 集成
- 开放 API
- 数据统计
